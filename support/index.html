Mailing list: Jurism news and support

Sign up for the Jurism support mailing list to receive project news and participate in discussions. The list is moderated (so no exciting investment appeals), and low-traffic (typically no more than two or three posts per week).

Documentation

Documentation for the project (ranging from user tutorials to technical documentation) is published on ReadTheDocs. in two locations:

  • Jurism Manual
  • citeproc-js Manual

Source code for the various moving parts that make up the Jurism ecosystem is divided across two locations:

Jurism Project
A GitHub "organization" housing forks of several Zotero repositories, a few Jurism-specific plugins, and tools used for maintaining and extending Jurism.
Legal Resource Registry
A GitHub project in my personal space for storing jurisdiction and court maps used by the Jurism client and the citation processor.
