Juris-M Essentials

Library Client
click for latest release

Juris-M is a variant of the Zotero research platform, with robust legal and multilingual support.

Juris-M is a rebranding of Multilingual Zotero (MLZ). It is maintained by the same person (@fgbjr), and it is still free software. What has changed is that Juris-M is now a “better neighbour” to its parent project Zotero.

The Juris-M Client will activate only after Zotero for Firefox has been disabled or removed. It operates on a copy of the previous database: existing Zotero data will not be affected. Standalone Zotero can be run on the same computer as Juris-M, and the two can share the same data if synced to the same account on zotero.org. (Juris-M will behave in the same way as Zotero, entering connector mode when Standalone is active.)

Jurisdiction Support
click for latest release

To enable modular legal style support in Juris-M, install the Myles plugin for Firefox. The plugin provides jurisdiction-specific style code to the citation processor, updated automatically at intervals from the Juris-M Style Modules repository.

If you would like to contribute code to the repository, begin by reviewing the interactive tutorial in the Style Development area of this website (you will be prompted to sign in with a GitHub account).

Note that the plugin only takes effect for the JM family of styles. Modular legal style support can be enabled in mainstream CSL and legacy MLZ styles with minimal effort, however. For details, feel free to ping me on Twitter (@fgbjr).

Abbreviation Filter
click for latest release

The Abbreviation Filter provides style-specific abbreviation lists for use with the Juris-M Client, including those set forth in The Bluebook: A Uniform System of Citation.

The Filter provides an interface for adding, editing and managing abbreviation lists, accessible under an Abbrevs. button that appears in the word processor citation popups of Juris-M and Zotero, and via the Open Style Editor pane of Juris-M. (The Filter will function in official Zotero, although it is not currently accessible in the Style Editor.)

The Filter allows legal resources to be given their full, library-friendly names in Juris-M data, removing some of the mystery for collaborators not schooled in law-specific abbreviation arcana.

RTF/ODF Scan
visit project website

For writing platforms not directly supported by a word processor plugin (Scrivener is a common case), Juris-M can be used with the RTF/ODF Scan plugin. The plugin enables scannable citation placeholders that can be pasted into any document text. The placeholders can be converted to “live” Zotero/Juris-M citations when the document is exported in the ODF format used by LibreOffice.

Bluebook Signals
visit project site

Law review editors in the United States generally require the use of a conventional set of “signals” to indicate the manner of support that individual citations lend to the author”s argument (to wit: e.g., accord, see, see also, cf., contra, but see, but cf., and see generally).

This plugin enables convenient shortcut entry of such signals.

CSL Processor and Zotero Gadgets

(For the curious, propachi is a Japanese slang term for a professional pachinko player.)

citeproc-js
visit project site

The citeproc-js CSL processor is the magic behind citation formatting in Zotero, Juris-M, Mendeley, and many other modern websites and research platforms. Like Juris-M, it is maintained by @fgbjr.

The processor code itself is primarily of interest to software developers.

Processor Patch
click for latest release

On rare occasions, issues arise that require a change in the core code of the citation processor. Zotero users who wish to take advantage of a fix immediately can do so by installing this propachi-vanilla plugin. Its only effect is to swap in the latest processor version, and it can be removed when a Zotero version incorporating the fix is released.

Uppercase Subtitles
click for latest release

Some citation styles require that the first character of a subtitle in “sentence case” be capitalized, and others do not. Official CSL does not currently have a mechanism for representing this difference, and citeproc-js will render subtitles as written. Zotero users who require uppercasing of the first character in subtitles can install this propachi-upper plugin to achieve the desired effect.

Citation Stylesheets

The Desktop Client is distributed with a full set of Juris-M citation styles. Use the links below for reinstallation and updates if required.

JM Chicago Full Note
(initial release)

This is the standard CSL Chicago Full Note style with minor modifications to add modular legal style support.

CMS Full Note with Bluebook-style referencing for legal resources is the recommended citation style for English-language theses in the Nagoya University Graduate School of Law, the current institutional home of Juris-M.

JM Baby Blue
(testing only)

This is a test release of preliminary code for styles that incorporate Juris-M modular legal style support. The code of this style will be finished out when the text of the Baby Blue style guide is release later this year.

With the deployment of the Juris-M client, the MLZ family of styles will be phased out, and replaced with JM styles that, like JM Baby Blue, rely upon a common base of style modules for references to primary legal materials.

MLZ Bluebook
(current)

Use this MLZ stylesheet for Bluebook referencing. It will be replaced within the next few months by a functionally equivalent Juris-M modular style to be released as "JM Baby Blue," when the eponymous style guide becomes available.

See Leslie Gordon, "Legal minds differ on whether The Bluebook is subject to copyright," ABA Journal, February 1, 2015 for information on the Baby Blue project, and on the cause of the upcoming change in the name of the stylesheet.

MLZ Chicago Full Note
(current)

The MLZ implementation of Chicago Manual of Style full note referencing.

This stylesheet will be superceded in Summer 2015 with a functionally equivalent version that supports modular legal referencing, to be released as "JM Chicago Full Note."

MLZ OSCOLA
(current)

The MLZ implementation of the Oxford Standard for the Citation of Legal Resources.

This stylesheet will be superceded in Summer 2015 with a functionally equivalent version that supports modular legal referencing, to be released as "JM OSCOLA."

MLZ New Zealand Law
(current)

The MLZ implementation of the New Zealand Law style for legal resources.

An early contribution to the MLZ family of styles from Wellington University, the modular upgrade of this stylesheet will follow that of OSCOLA, which its cite forms substantially resemble.

CSL Style Repository
from Zotero

The Juris-M citation processor is compatible with mainstream CSL, and with some small exceptions, mainstream CSL styles should render equally well in Juris-M.

Legal referencing support can be added to a mainstream CSL style without exceptional effort. If you require this, please post a note to @fgbjr on Twitter.
