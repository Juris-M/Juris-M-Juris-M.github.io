Core Plugins

Desktop Client
version 4.0.26m512

A variant of the popular Zotero reference manager, with support for legal materials and multilingual citations. The Juris-M Desktop Client is a Firefox plugin, and will install automatically when the link is clicked from Firefox.

Please note that the plugin will modify an existing Zotero database, and that after installation the database cannot be accessed with Zotero Standalone on the same computer.

This is a simple rebranding of Multilingual Zotero (MLZ), with no major code changes, and is still maintained by Frank Bennett.

Abbreviation Filter
version 1.0.200

Style-specific abbreviation lists for use with the Juris-M Desktop Client. This is also a Firefox plugin, and will install automatically when the link is clicked from Firefox.

Abbreviation entries can be added and edited from within a document, or through the CSL Editor accessed from the Advanced tab of the Preferences popup in the Desktop Client.

Jurisdiction Support
version 1.1.1

Jurisdiction modules submitted through the Style Editor are distributed through this Firefox plugin. When the plugin is installed, jurisdiction-specific formatting is applied automatically. Style refinements and additions are distributed as automatic updates to the plugin—it is is literally plug-and-play.

Note that the plugin only takes effect for (still forthcoming) Juris-M styles. The legacy MLZ styles do not have modular style support.

Optional Plugins

WP Integration
from Zotero

Most users will choose to install one of the word processor integration plugins offered by the Zotero team, which provide the same functionality enjoyed by users of mainstream Zotero.

No special configuration is required to use the Zotero word processor plugins with Juris-M.

RTF/ODF Scan
version 1.0.22

This plugin provides robust citation support for word processing platforms other than Word and LibreOffice (which are supported directly by the WP Integration plugins above).

Using this plugin, citation placeholders can be pasted into any document text. If the document is then converted to Open Document Format (ODF), the placeholders can be transformed into "live" citations, after which LibreOffice can apply citation styles to the document in the usual way.

Bluebook Signals
version 1.0.5

A leading citation guide for legal writing in the United States specifies a set of "signals," used to indicate the relation of each cited source to the author's line of argument.

This plugin enables convenient shortcut entry of citation signals.

Citation Stylesheets

The Desktop Client is distributed with a full set of Juris-M citation styles. Use the links below for reinstallation and updates if required.

MLZ Bluebook
(current)

Use this MLZ stylesheet for Bluebook referencing. It will be replaced within the next few months by a functionally equivalent Juris-M modular style to be released as "JM Baby Blue," when the eponymous style guide becomes available.

See Leslie Gordon, "Legal minds differ on whether The Bluebook is subject to copyright," ABA Journal, February 1, 2015 for information on the Baby Blue project, and on the cause of the upcoming change in the name of the stylesheet.

MLZ Chicago Full Note
(current)

The MLZ implementation of Chicago Manual of Style full note referencing.

This stylesheet will be superceded in Summer 2015 with a functionally equivalent version that supports modular legal referencing, to be released as "JM Chicago Full Note."

MLZ OSCOLA
(current)

The MLZ implementation of the Oxford Standard for the Citation of Legal Resources.

This stylesheet will be superceded in Summer 2015 with a functionally equivalent version that supports modular legal referencing, to be released as "JM OSCOLA."

MLZ New Zealand Law
(current)

The MLZ implementation of the New Zealand Law style for legal resources.

An early contribution to the MLZ family of styles from Wellington University, the modular upgrade of this stylesheet will follow that of OSCOLA, which its cite forms substantially resemble.

CSL Style Repository
from Zotero

The Juris-M citation processor is compatible with mainstream CSL, and with some small exceptions, mainstream CSL styles should render equally well in Juris-M.

Legal referencing support can be added to a mainstream CSL style without exceptional effort. If you require this, please post a note to @fgbjr on Twitter.
